Users

  • id (INTEGER AUTOINCREMENT)
  • name (STRING)
  • email (STRING)
  • password (STRING)
  • remember_token (XXX)
  • timestamps (XXX)
  • 1 user heeft meerdere items (foreign key in Items)
  • 1 user heeft meerdere biedingen (foreign key in Biddings)

Items

  • id (INTEGER AUTOINCREMENT)
  • lot_id (lot nummer -> auto incremented value?) (INTEGER)
  • category (STRING)
  • auction_title (STRING)
  • year (DATE)
  • width (in cm) DECIMAL(4,2)
  • height (in cm) DECIMAL(4,2)
  • depth (in cm) DECIMAL(4,2)
  • description (TEXT)
  • condition (TEXT)
  • origin (STRING)
  • artwork_image (BLOB)
  • signature_image (BLOB)
  • optional_image (BLOB)
  • minimum_estimated_price (DECIMAL(12,2))
  • maximum_estimated_price (DECIMAL(12,2))
  • buyout_price (DECIMAL(12,2))
  • end_date (DATETIME)
  • agreed (BOOLEAN)
  • amount_of_bids (standaard 0) (INTEGER)
  • 1 item heeft meerdere biedingen (foreign key in Biddings)
  • fk_user_id (FK INTEGER)
  • fk_watchlist_id (FK INTEGER)

Biedingen (Biddings)

  • id (INTEGER AUTOINCREMENT)
  • fk_user_id (FK INTEGER)
  • fk_item_id (FK INTEGER)

Watchlists

  • id (INTEGER AUTOINCREMENT)
  • 1 watchlist heeft meerdere items (foreign key in items)
  • fk_user_id (FK INTEGER)
